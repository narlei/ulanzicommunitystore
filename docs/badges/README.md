# Ulanzi Community Store badges

Use these badges in your plugin README to show that the plugin is available on the **Ulanzi Community Store**.

## Preview

### Brand badge (recommended)

[![Available on Ulanzi Community Store](ulanzi-community-store.svg)](https://ulanzicommunitystore.narlei.com)

### Shield style

[![Ulanzi Community Store](ulanzi-community-store-shield.svg)](https://ulanzicommunitystore.narlei.com)

## Copy-paste for your README

### Brand badge

```markdown
[![Available on Ulanzi Community Store](https://raw.githubusercontent.com/narlei/ulanzicommunitystore/main/docs/badges/ulanzi-community-store.svg)](https://ulanzicommunitystore.narlei.com)
```

### Shield style

```markdown
[![Ulanzi Community Store](https://raw.githubusercontent.com/narlei/ulanzicommunitystore/main/docs/badges/ulanzi-community-store-shield.svg)](https://ulanzicommunitystore.narlei.com)
```

### shields.io (no asset)

If you prefer not to hotlink an SVG from this repo:

```markdown
[![Ulanzi Community Store](https://img.shields.io/badge/Ulanzi%20Community%20Store-available-41e6c3?style=flat&labelColor=1f2937)](https://ulanzicommunitystore.narlei.com)
```

## Website URLs (after marketing deploy)

Same files are also published on the marketing site:

```markdown
[![Available on Ulanzi Community Store](https://ulanzicommunitystore.narlei.com/assets/badge.svg)](https://ulanzicommunitystore.narlei.com)
```

```markdown
[![Ulanzi Community Store](https://ulanzicommunitystore.narlei.com/assets/badge-shield.svg)](https://ulanzicommunitystore.narlei.com)
```

Prefer the `raw.githubusercontent.com` URLs above if you want the badge to work even when the marketing site is unavailable; use the website URLs for a shorter branded host.

## Tips

- Prefer the **brand badge** when you want the store icon and wording to stand out.
- Prefer the **shield** when you already have a row of shields.io badges.
- Link to [ulanzicommunitystore.narlei.com](https://ulanzicommunitystore.narlei.com) (or your plugin page on the site when available).
- Only use the badge after your plugin is listed in the [community registry](../../registry/plugins).
