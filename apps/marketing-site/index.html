Unofficial desktop app

Install Ulanzi Deck and Dial plugins without hunting through repos.

Download latest release Publish a plugin

macOS · Windows · Free and open source

One-click installs

The app downloads the latest plugin release and places it in the Ulanzi plugins folder for you.

Catalog-first

Plugins come from the official registry, generated automatically from GitHub releases and manifests.

Cross-platform

Electron builds are published for macOS and Windows through GitHub Releases.

How it works

  1. 1

    Download the app

    Grab the latest release for macOS or Windows straight from GitHub.

  2. 2

    Browse the catalog

    See every approved plugin with its description, version, and source repository.

  3. 3

    Install, update, or remove

    One click writes the plugin into your Ulanzi plugins folder — no manual unzip, no path juggling.
