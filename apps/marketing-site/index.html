Ulanzi Plugin Store Download

Unofficial desktop app

Install Ulanzi Deck and Dial plugins without hunting through repos.

Browse the catalog, install plugins, update them, and remove them from a focused desktop app for macOS and Windows.

Download latest release Publish a plugin

One-click installs

The app downloads the latest plugin release and places it in the Ulanzi plugins folder.

Catalog-first

Plugins come from the official registry, generated from GitHub releases and manifests.

Cross-platform

Electron builds are published for macOS and Windows through GitHub Releases.

Independent project

This is an unofficial community project. It is not affiliated with, endorsed by, or maintained by Ulanzi.
