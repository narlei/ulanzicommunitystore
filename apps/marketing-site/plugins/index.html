Recommended

Install via Terminal

Detects your OS and installs the right build. On macOS, files fetched with curl aren't quarantined the way browser downloads are, so Gatekeeper never blocks the app.

curl -fsSL https://raw.githubusercontent.com/narlei/ulanzicommunitystore/main/install.sh | bash

Recommended

Install via PowerShell

Native Windows path — no bash required. Downloads the latest .exe and launches the installer.

irm https://raw.githubusercontent.com/narlei/ulanzicommunitystore/main/install.ps1 | iex

The .dmg triggers a normal macOS "unidentified developer" warning on first launch, since the app isn't Apple-notarized (no paid developer account). That's expected — see the README for how to open it, or use the Terminal install above instead.

Run the downloaded .exe and complete the installer wizard. Prefer the PowerShell one-liner above if you want a one-step setup.
