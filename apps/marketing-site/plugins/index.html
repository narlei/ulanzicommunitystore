Install via Terminal

macOS

Skips the Gatekeeper security prompt entirely — files fetched with curl aren't quarantined the way browser downloads are, so the app opens right away.

curl -fsSL https://raw.githubusercontent.com/narlei/ulanzicommunitystore/main/install.sh | bash

🍎 macOS .dmg 🪟 Windows .exe All releases →

The .dmg triggers a normal macOS "unidentified developer" warning on first launch, since the app isn't Apple-notarized (no paid developer account). That's expected — see the README for how to open it, or just use the Terminal install above instead.
